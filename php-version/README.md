# Campus Trip PHP Version (Laravel)

This folder will contain the Laravel-based PHP rewrite of your Campus Trip app.

## Why Laravel?
- Modern, secure, and scalable PHP framework
- Blade templating for UI
- Built-in routing, controllers, and authentication
- Easy API integration

## Next Steps
1. Initialize a new Laravel project here (run: composer create-project laravel/laravel .)
2. Copy .env.example to .env and set your API keys (Supabase, LOVABLE_API_KEY, etc.)
3. Implement routes, controllers, and Blade views to match your React app's pages
4. Use Tailwind CSS for styling (Laravel supports it out of the box)

**Note:**
- All API endpoints and keys from your main project will be referenced here.
